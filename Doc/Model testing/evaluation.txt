# Evaluation Module (evaluation.db)

## Description

The Evaluation Module is responsible for storing, managing, and analyzing the performance results of the Crime Rate Prediction System. This module records prediction outputs, model performance metrics, accuracy scores, error values, and testing results for future reference and comparison.

The evaluation database serves as a repository for model assessment and helps determine the effectiveness of the trained Machine Learning and Deep Learning models.

---

# Features

## Prediction Result Storage

- Stores predicted crime rates.
- Maintains historical prediction records.
- Saves state-wise prediction outputs.

## Performance Evaluation

- Records model accuracy.
- Stores error metrics.
- Tracks model performance over time.

## Testing Analysis

- Compares actual and predicted values.
- Measures forecasting quality.
- Generates evaluation reports.

## Database Management

- Secure data storage.
- Fast retrieval of evaluation records.
- Efficient performance tracking.

# Mathematical Formulas

## Mean Squared Error (MSE)

Measures average squared prediction error.

\[
MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2
\]

Where:

- \(y_i\) = Actual crime value
- \(\hat{y_i}\) = Predicted crime value
- \(n\) = Total observations

---

## Root Mean Squared Error (RMSE)

Measures prediction accuracy.

\[
RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2}
\]

Lower RMSE indicates better model performance.

---

## Mean Absolute Error (MAE)

Measures average prediction deviation.

\[
MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i-\hat{y_i}|
\]

---

## Accuracy Formula

\[
Accuracy = \frac{Correct\ Predictions}{Total\ Predictions}\times100
\]

# Common Errors

## Error 1: Database File Not Found

### Error Message

```python
sqlite3.OperationalError
```

### Cause

Evaluation database file is missing.

### Solution

Verify:

```text
evaluation.db
```

exists in the project directory.
