Crime Wise Prediction Module (crime_wise_prediction.py)
Description
The crime_wise_prediction.py module is responsible for analyzing and predicting crime categories individually. Unlike state-wise prediction, this module focuses on specific crime types such as Murder, Theft, Robbery, Kidnapping, Assault, and other IPC crimes. The module enables users to select a state and crime category, analyze historical trends, generate future predictions, and visualize crime patterns through graphical representations.

Features
Crime Category Selection
Select individual crime categories.

Analyze crime-specific trends.

View historical crime records.

State Selection
Select a state or union territory.

Generate crime-wise forecasts for the selected state.

Crime Prediction
Predict future occurrences of specific crime categories.

Display prediction results for upcoming years.

Analyze crime growth and decline trends.

Historical crime trend graphs.

Future prediction graphs.

Comparative crime charts.

Crime category-wise analysis.

Statistical trend evaluation.

Pattern identification.

Common Errors
Error 1: State Not Selected
The error message is Please select a state first. This occurs because the user attempted to generate a prediction without selecting a state. Solve this by selecting a state from the dropdown menu before proceeding.

Error 2: Crime Category Not Selected
The error message is Please select a crime category. This happens when a crime type is not selected before clicking Predict. Choose a crime category from the list to resolve this.

Error 3: Dataset File Not Found
The error message is FileNotFoundError. This indicates that the crime dataset path is incorrect or the file is missing. Verify the dataset location and update the file path in the code.

Error 4: Graph Not Displaying
This happens if the graph data is empty or invalid. To fix this, verify your dataset values, check your graph plotting code, and ensure that Matplotlib is installed correctly.

Error 5: Prediction Generation Failed
This is caused by insufficient data available for the selected crime category. Ensure the dataset is complete, check your preprocessing steps, and verify that there are enough historical records to generate a prediction.

Debugging Guide
To debug your system, use these print statements to verify operations:

Use print(Crime Dataset Loaded Successfully) to check data loading.

Use print(Selected State: state) and print(Selected Crime: crime_type) to check user inputs.

Use print(Crime Prediction Started) and print(Crime Prediction Completed) to monitor the process.

Use print(Generating Graph) and print(Graph Generated Successfully) to check visualizations.

Use print(predicted_values) to verify that the model is producing valid output.

Workflow
The user opens the crime-wise prediction module and selects a state, followed by a specific crime category. The system loads the historical data, generates a prediction, displays the results, and creates a graph to show the forecast output.
