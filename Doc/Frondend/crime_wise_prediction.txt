# Crime Wise Prediction Module (crime_wise_prediction.py)

## Description

The `crime_wise_prediction.py` module is responsible for analyzing and predicting crime categories individually. Unlike state-wise prediction, this module focuses on specific crime types such as Murder, Theft, Robbery, Kidnapping, Assault, and other IPC crimes.

The module enables users to select a state and crime category, analyze historical trends, generate future predictions, and visualize crime patterns through graphical representations.

---

# Features

## Crime Category Selection

- Select individual crime categories.
- Analyze crime-specific trends.
- View historical crime records.

## State Selection

- Select a state or union territory.
- Generate crime-wise forecasts for the selected state.

## Crime Prediction

- Predict future occurrences of specific crime categories.
- Display prediction results for upcoming years.
- Analyze crime growth and decline trends.

## Graph Visualization

- Historical crime trend graphs.
- Future prediction graphs.
- Comparative crime charts.

## Data Analysis

- Crime category-wise analysis.
- Statistical trend evaluation.
- Pattern identification.

# Common Errors

## Error 1: State Not Selected

### Error Message

```text
Please select a state first
```

### Reason

The user attempted to generate a prediction without selecting a state.

### Solution

Select a state from the dropdown menu before proceeding.

---

## Error 2: Crime Category Not Selected

### Error Message

```text
Please select a crime category
```

### Reason

A crime type was not selected before prediction.

### Solution

Choose a crime category before clicking Predict.

---

## Error 3: Dataset File Not Found

### Error Message

```text
FileNotFoundError
```

### Reason

The crime dataset path is incorrect or the file is missing.

### Solution

Verify the dataset location and update the file path.

---

## Error 4: Graph Not Displaying

### Reason

The graph data is empty or invalid.

### Solution

- Verify dataset values.
- Check graph plotting code.
- Ensure Matplotlib is installed correctly.

---

## Error 5: Prediction Generation Failed

### Reason

Insufficient data available for the selected crime category.

### Solution

- Verify dataset completeness.
- Check preprocessing steps.
- Ensure enough historical records exist.
