State Wise Prediction Module (state_wise_prediction.py)
Description
The state_wise_prediction.py module is responsible for predicting future crime rates for individual states and union territories of India. The module analyzes historical crime data, identifies trends, and generates future crime forecasts using Machine Learning and Deep Learning techniques. Users can select a state, view historical crime records, generate future predictions, and visualize crime trends through interactive graphs.

View state-specific crime statistics.

Predict crime rates for future years.

Generate state-specific forecasts.

Display prediction results instantly.

Store generated predictions.

Common Errors
Error 1: State Not Selected
The error message is Select a state. This happens when the user attempts to generate predictions without selecting a state first. Solve this by selecting a state from the dropdown list before clicking the Predict button.

Error 2: Dataset Not Found
The error message is FileNotFoundError. This means the crime dataset file is missing or the path is incorrect. Verify that the dataset path DATA_PATH = "data/crime_data_india.csv" is correct and that the file exists.

Error 3: Dataset Loading Failed
The error message is Unable to load dataset. This is caused by a corrupted or improperly formatted CSV file. Verify the CSV structure, check column names, and remove any corrupted rows.

Error 4: Prediction Not Generated
This occurs if the selected state does not contain sufficient historical data. Verify state data availability, check dataset completeness, and ensure all required columns exist.

Error 5: Graph Not Displaying
This happens if prediction data is empty or graph generation fails. Verify the prediction values, check that Matplotlib is installed, and ensure that the graph plotting functions execute correctly.

Workflow
The user opens the state-wise prediction module and selects a state. The system loads historical crime data and analyzes crime trends. It then generates a future prediction, stores that prediction in the database, generates a graph, and displays the final results to the user.
