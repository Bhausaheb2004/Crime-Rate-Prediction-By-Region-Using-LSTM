# State Wise Prediction Module (state_wise_prediction.py)

## Description

The `state_wise_prediction.py` module is responsible for predicting future crime rates for individual states and union territories of India. The module analyzes historical crime data, identifies trends, and generates future crime forecasts using Machine Learning and Deep Learning techniques.

Users can select a state, view historical crime records, generate future predictions, and visualize crime trends through interactive graphs.

---

# Features

## State Selection

- Select any available State/UT.
- Load historical crime records.
- View state-specific crime statistics.

## Historical Crime Analysis

- Analyze past crime trends.
- Display year-wise crime data.
- Identify increasing or decreasing crime patterns.

## Future Crime Prediction

- Predict crime rates for future years.
- Generate state-specific forecasts.
- Display prediction results instantly.

## Data Visualization

- Historical crime trend graph.
- Future crime prediction graph.
- Comparative trend analysis.

## Database Integration

- Store generated predictions.
- Maintain prediction history.
- Retrieve previous prediction records.

# Common Errors

## Error 1: State Not Selected

### Error Message

```text
Select a state
```

### Reason

User attempted to generate predictions without selecting a state.

### Solution

Select a state from the dropdown list before clicking the Predict button.

---

## Error 2: Dataset Not Found

### Error Message

```text
FileNotFoundError
```

### Reason

The crime dataset file is missing or the dataset path is incorrect.

### Solution

Verify the dataset path:

```python
DATA_PATH = "data/crime_data_india.csv"
```

and ensure the file exists.

---

## Error 3: Dataset Loading Failed

### Error Message

```text
Unable to load dataset
```

### Reason

The CSV file may be corrupted or improperly formatted.

### Solution

- Verify CSV structure.
- Check column names.
- Remove corrupted rows.

---

## Error 4: Prediction Not Generated

### Reason

The selected state does not contain sufficient historical data.

### Solution

- Verify state data availability.
- Check dataset completeness.
- Ensure required columns exist.

---

## Error 5: Graph Not Displaying

### Reason

Prediction data is empty or graph generation failed.

### Solution

- Verify prediction values.
- Check Matplotlib installation.
- Ensure graph plotting functions execute correctly.
