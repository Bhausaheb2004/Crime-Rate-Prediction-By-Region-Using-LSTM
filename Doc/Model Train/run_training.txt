# Model Training Execution Module (run_training.py)

## Description

The `run_training.py` file is responsible for executing the complete model training process of the Crime Rate Prediction System. This module loads the historical crime dataset, preprocesses the data, trains the LSTM model, evaluates its performance, and saves the trained model for future crime prediction.

It acts as the main training engine of the project and is typically executed whenever the model needs to be retrained with updated crime records.

---

# Features

## Dataset Loading

- Loads crime dataset from CSV files.
- Validates dataset structure.
- Reads historical crime records.

## Data Preprocessing

- Handles missing values.
- Removes duplicate records.
- Performs normalization.
- Converts data into time-series format.

## Model Training

- Trains LSTM Neural Network.
- Learns historical crime patterns.
- Generates forecasting model.

## Model Evaluation

- Calculates training accuracy.
- Generates loss metrics.
- Measures prediction performance.

## Model Saving

- Saves trained model files.
- Stores model weights.
- Enables future prediction without retraining.

# Common Errors

## Error 1: Dataset Not Found

### Error Message

```python
FileNotFoundError
```

### Cause

Dataset file path is incorrect.

### Solution

Verify:

```python
crime_data_india.csv
```

exists in the data folder.

---

## Error 2: TensorFlow Not Installed

### Error Message

```python
ModuleNotFoundError: No module named 'tensorflow'
```

### Cause

TensorFlow library is missing.

### Solution

Install TensorFlow:

```bash
pip install tensorflow
```

---

## Error 3: Insufficient Data

### Error Message

```python
ValueError
```

### Cause

Dataset does not contain enough records for training.

### Solution

Use larger historical crime datasets.

---

## Error 4: Model Saving Failed

### Error Message

```python
PermissionError
```

### Cause

Application cannot write model files.

### Solution

Verify save directory permissions.

---

## Error 5: Memory Error

### Error Message

```python
MemoryError
```

### Cause

Large dataset exceeds system memory.

### Solution

Reduce batch size or use a machine with more RAM.
