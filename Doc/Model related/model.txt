# LSTM Theory

Long Short-Term Memory (LSTM) is a specialized Recurrent Neural Network (RNN) designed to handle sequential and time-series data.

Unlike traditional neural networks, LSTM can remember information over long periods and effectively learn temporal dependencies.

This makes LSTM highly suitable for crime forecasting where future crime rates depend on historical crime patterns.

# Mathematical Formulas

## Forget Gate

Determines which information should be discarded.

\[
f_t = \sigma(W_f[h_{t-1},x_t]+b_f)
\]

Where:

- \(f_t\) = Forget gate output
- \(W_f\) = Weight matrix
- \(h_{t-1}\) = Previous hidden state
- \(x_t\) = Current input
- \(b_f\) = Bias

---

## Input Gate

Determines what new information should be stored.

\[
i_t = \sigma(W_i[h_{t-1},x_t]+b_i)
\]

---

## Candidate State

Creates new candidate values.

\[
\tilde{C_t}=tanh(W_c[h_{t-1},x_t]+b_c)
\]

---

## Cell State Update

\[
C_t=f_t*C_{t-1}+i_t*\tilde{C_t}
\]

---

## Output Gate

\[
o_t=\sigma(W_o[h_{t-1},x_t]+b_o)
\]

---

## Hidden State

\[
h_t=o_t*tanh(C_t)
\]

---

# Loss Function

## Mean Squared Error (MSE)

\[
MSE=\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2
\]

Where:

- \(y_i\) = Actual value
- \(\hat{y_i}\) = Predicted value

The objective is to minimize MSE during training.

## Error 3: Input Shape Mismatch

### Error Message

