# Prediction Module (predict.py)

## Description

The Prediction Module is responsible for generating future crime forecasts using the trained LSTM models. This module loads the saved model, processes user-selected state data, performs prediction, and displays future crime trends through graphical visualizations and statistical outputs.

The module acts as the final stage of the Crime Rate Prediction System, where trained models are utilized to estimate future crime rates based on historical crime records.

---

# Objectives

- Load trained LSTM models.
- Generate future crime predictions.
- Forecast state-wise crime rates.
- Display prediction results.
- Visualize future crime trends.
- Support decision-making through predictive analytics.

# Mathematical Formula

## LSTM Prediction Formula

\[
Prediction = LSTM(Historical\ Crime\ Data)
\]

Where:

- Historical crime data serves as input.
- Trained LSTM model processes patterns.
- Future crime rate is generated as output.

---

## Mean Squared Error (MSE)

\[
MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2
\]

Used to measure prediction error.

---

## Root Mean Squared Error (RMSE)

\[
RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2}
\]

Lower RMSE indicates higher prediction accuracy.

# Common Errors

## Error 1: Model File Not Found

### Error Message

```python
FileNotFoundError
```

### Cause

Trained model file is missing.

### Solution

Verify model location:

```text
saved_models/
```

---

## Error 2: State Model Not Found

### Error Message

```python
FileNotFoundError
```

### Cause

Selected state's model has not been trained.

### Solution

Train the state model before prediction.

---

## Error 3: Dataset Not Found

### Error Message

```python
FileNotFoundError
```

### Cause

Crime dataset path is incorrect.

### Solution

Verify:

```text
data/crime_data_india.csv
```

exists.

---

## Error 4: Invalid State Selection

### Error Message

```python
ValueError
```

### Cause

Selected state does not exist in dataset.

### Solution

Verify available states:

```python
print(df["STATE/UT"].unique())
```

---

## Error 5: Empty Prediction Data

### Error Message

```python
ValueError
```

### Cause

No historical records available.

### Solution

Verify dataset contains sufficient state data.

---

## Error 6: Model Loading Error

### Error Message

```python
OSError
```

### Cause

Corrupted model file.

### Solution

Retrain and save the model again.

---

# Error Handling

The prediction module uses exception handling to prevent application crashes.

```python
try:
    prediction = model.predict(data)

except FileNotFoundError:
    print("Model file not found")

except ValueError:
    print("Invalid prediction input")

except Exception as e:
    print("Prediction Error:", e)
```
